Managing Mods#

Skyve

  • A mod for managing other mods
  • Download from the official Mod platform and restart the game. There will be an Install Skyve option, which allows you to install Skyve
  • Before starting the game next time, open Skyve to check for mod updates before playing

Prerequisite Mods#

  • Unified Icon Library
  • Asset Icon Library
  • Extra Lib
  • I18n EveryWhere

Recommended#

  • Move It

    • Reference video: 都市天际线2~神级模组move it重大更新！_实况解说

  • Find It

    • Directly search for various elements you want
    • Don’t use Find It to place vehicles, as it will cause pathfinding and other bugs
    • If you place buildings without designating functional zones, buildings will become abandoned. Use it together with the Plop the Growables mod

  • Better Bulldozer

    • A better bulldozer tool

  • Anarchy

    • No collision

  • Traffic

    • Road connections

  • Traffic Lights Enhancement Alpha

    • Traffic light control

  • Extended Tooltip

    • When hovering your mouse over characters and buildings, displays very detailed information

  • Realistic Parking Mod

    • Realistic parking mod that adds more parking spaces to buildings

  • First Person Camera Continued

    • First-person perspective to explore your city

  • Zone Organizer

    • After subscribing to a large number of asset packs, this keeps low, medium, and high density zones organized

  • Road Builder

    • Allows you to customize roads
    • Currently in Beta version, consider adding once it becomes stable
    • Reference video: 都市天际线2神级模组——road builder震撼来袭！_哔哩哔哩_bilibili

Optional#

  • Extra Landscaping Tools

    • Custom brush shapes
    • Add natural resources

  • Water Features

    • Water source tools

  • Extra Detailing Tools

    • More beautification asset tools, terrain modification, and decal placement

  • Asset Packs Manager

    • Asset import mod
    • May be disabled when the asset editor comes online

  • Extra Assets Importer

    • Beautification prerequisite component for importing assets
    • May be disabled when the asset editor comes online

  • Area Bucket

    • Terrain switching mod
    • Fill terrain

  • Asset Variation Changer

    • Standardize decoration colors

  • Recolor

    • Color adjustment

  • Road Wear Remover

    • Remove road tire marks

  • Advanced Line Tool

    • Custom placement of objects, such as placing park benches in a line, circle, etc.

  • Magic Taxi

    • Makes all taxis entering the city from outside disappear
    • Does not affect local taxi companies

  • No Dogs

    • No dogs spawning

  • Hall of Fame

    • Take screenshots, upload to server, city exhibitions
    • The background image in the main menu will be replaced with background images from other players

  • Tree Controller

  • Static Ploppables

    • Asset decorations without attributes, for scenery builders. Only buildings without residents, no practical function

  • Hide Buildings Notification

    • Hide icons on top of buildings
